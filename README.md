# Exercícios RI2

Repositório com a resolução do exercício de Técnicas de Programação RI2, utilizando JavaScript.

*P.S.: No arquivo RIs existe um easter egg.*

## 📋 Pré-requisitos

Para rodar o projeto, você precisará do Node.js instalado na sua máquina.

### 1. Instalar o Node.js (Windows)
```bash
winget install OpenJS.NodeJS.LTS
```

### 2. Como inicializar os projetos
Para executar diretamente pelo terminal, use o comando node seguido do nome do arquivo:
```Bash
node nomeDoArquivo.js
```
(Exemplo: node RI2.js)

### 4. Descrição dos Exercícios
RI2: Cadastro Base

A ideia é adicionar os dados de Cliente, Telefone e Endereço de uma empresa não especificada.
